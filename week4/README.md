# Week 4 – SQL Practice (Employees Table)

This document contains SQL practice exercises using the **employees** table.

1. Display `first_name` and `last_name` using the aliases **"First Name"** and **"Last Name"`.

2. Retrieve the unique `department_id` values from the `employees` table.

3. Retrieve all employee details and sort them by `first_name` in descending order.

4. Display `first_name`, `last_name`, `salary`, and **PF** for all employees, where **PF is 15% of salary**.

5. Display `employee_id`, `first_name`, `last_name`, and `salary`, sorted by salary in ascending order.

6. Retrieve the total salary that must be paid to all employees.

7. Retrieve the maximum salary and minimum salary from the `employees` table.

8. Retrieve the average salary and the total number of employees from the `employees` table.

9. Retrieve the total number of employees in the company.

10. Retrieve the number of distinct job roles available in the `employees` table.

11. Retrieve `first_name` from the `employees` table and display it in uppercase.

12. Retrieve the first 3 characters of `first_name` from the `employees` table.

13. Write a query to calculate the following expression:

   171 × 214 + 625

14. Retrieve the full name by combining `first_name` and `last_name`.

   Example output:
   Ellen Abel  
   Sundar Ande

15. Retrieve `first_name` from `employees` and remove any leading and trailing spaces.

16. Retrieve the length of `first_name` and `last_name` from the `employees` table.

17. Check if there are any employees whose `first_name` contains numeric characters.

18. Retrieve the first 10 rows from the `employees` table.

19. Retrieve the monthly salary for all employees.  
    Assume that `salary` stored in the database represents the annual salary, and the result should be rounded to 2 decimal places.
